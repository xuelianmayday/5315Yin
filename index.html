What is Docker

Docker is a software containerization platform. It performs operating system level virtulization - containerization. Docker is developed mainly for Linux, and uses the resource isolation features of the Linux kernel and a file system to enable independent "containers" to run within a single Linux instance, which prevents the overhead of maintaining virtual machines.

https://www.docker.com/what-docker
https://en.wikipedia.org/wiki/Docker_(software)

What is Nginx

NGINX is an open source software for web serving. It can also be used for reverse proxying, caching, media streaming, etc. It can be used to serve dynamic HTTP content on the network using FastCGI, SCGI handlers for scripts, WSGI application servers or Phusion Passenger modules, and it can also serve as a software load balancer. NGINX improves content and application delivery, improves security, facilitates availability and scalability for the busiest web sites on the Internet.

https://en.wikipedia.org/wiki/Nginx
https://www.nginx.com/resources/glossary/nginx/

What is VM (virtual machine)

Virtual machine is an emulation of a computer system. Virtual machines provide functionality of a physical computer. Their implementations may involve specialized hardware, software, or a combination. Some of virtual machines are used to emulate different architectures and allow software applications to excute and operating systems written for another CPU or architecture.

https://en.wikipedia.org/wiki/Virtual_machine

Who is Cloud Computing Service Provider - Digitalocean

DigitalOcean is a company that provides cloud infrastructure for developers and businesses. It provides an effective cloud computing platform of virtual servers and object storage, so that they can deploy and scale applications that run simultaneously on multiple computers.
